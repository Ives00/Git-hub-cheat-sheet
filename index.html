Welcome!

Hi new GitHub users! Welcome to my cheatsheet made for you! This is after creating a GtiHub account and applicable to Mac computers at the current moment.

    Installing Git via Terminal

  • Search and open Terminal
  • Type code: Git version
  • If installed, the version information will show. If not, a prompt will pop up for installation.

    • Initilizing First and Adding Files

  • Use "cd" and pull file into Terminal. This will direct to the Repo.
  • "git init" will initialize the Repo
  • Use "git and" drag file name.
  • Run "git status" to see what was changes were done

    • Committing Changes

  • Commit is a mark of progress.
  • When you make a new significant change then you need to commit it to Github.
  • Commit Early and commit often! The command: git commit -am "first commit" and press enter

    • Master to Main Branch

  • Branch-an independent line of development
  • Main - what developers are now using instead of "master"
  • Command to make the change: git branch -M main

    • Connecting Folder to GitHub

  • Need: adding the remote repo as an "origin" of the branch
  • Git Hub will give you the link...copy it (command + C)
  • Command: git remote add origin - paste link

    • Push Commits to GitHub

  • Once done making changes: need to commit them.
  • Use command -git push origin main
  • May ask to input username. You would also need a personal token key (done in developer settings (used as password).

    • Pulling Changes Made on Other Device

  • This is for when you make a change from different devices
  • To use changes on original files then you need to pull from GitHub
  • Use command: git pull origin main
  • This asks GitHub for changes made elsewhere.
