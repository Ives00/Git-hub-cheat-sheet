    Installing Git via Terminal

  • Search and open Terminal
  • Type code: Git version
  • If installed, the version information will show. If not, a prompt will pop up for installation.

    • Initilizing First and Adding Files

  • Use "cd" and pull file into Terminal. This will direct to the Repo.
  • "git init" will initialize the Repo
  • Use "git and" drag file name.
  • Run "git status" to see what was changes were done

    • Committing Changes

  • Commit is a mark of progress.
  • When you make a new significant change then you need to commit it to Github.
  • Commit Early and commit often! The command: git commit -am "first commit" and press enter
